# Digital-Modulation-and-Demodulation-Lab-Reports-TP1-TP2-TP3
Three practical laboratory reports covering digital baseband transmission, single-symbol digital modulation (ASK, PSK, CPFSK), and multi-symbol QPSK modulation and demodulation. All experiments were carried out using the Rohde &amp; Schwarz RTB2004 oscilloscope and dedicated hardware modules (MCM31, CE kit, ADALM-Pluto SDR).
